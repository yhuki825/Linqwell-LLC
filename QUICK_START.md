# 🚀 Linqwell LLC ウェブサイト - クイックスタート

## 📦 ファイル内容

- **linqwell-simple.html** - ウェブサイト本体（これ1つですべて）
- **linqwell-editing-guide.md** - 詳細な編集方法ガイド

## ⚡ 最速で始める3ステップ

### ステップ1: ファイルを開く
```
linqwell-simple.html をテキストエディタで開く
（メモ帳、VSCode、Sublime Textなど）
```

### ステップ2: テキストを編集
```
会社名、説明文、サービス内容などを変更
（HTMLタグには触らない）
```

### ステップ3: Cloudflare Pagesにアップロード
```
1. https://dash.cloudflare.com/ にアクセス
2. Pages → Create a project → Direct upload
3. linqwell-simple.html をアップロード
4. 完了！自動的にHTTPSで公開
```

## 🎯 よく変更する部分（TOP 5）

| 項目 | 場所 | 例 |
|------|------|-----|
| **会社名** | ナビゲーション、フッター | `Linqwell` → `あなたの会社名` |
| **メインタイトル** | ヒーローセクション | `世界と日本をリンクさせる...` |
| **説明文** | ヒーローセクション下 | `合同会社Linqwellは...` |
| **サービス内容** | 4つのカード | 医療機器、コンサルティング等 |
| **会社情報** | 会社概要セクション | 住所、代表者名等 |

## 🎨 色を変える

ファイルを開いて `Ctrl+H`（検索・置換）:

```
検索: #2563eb
置換: #ff6b6b  （赤色に変更する例）
すべて置換
```

## 📱 確認方法

1. ファイルを保存
2. ブラウザで `linqwell-simple.html` を開く
3. スマートフォンでも確認（自動対応）

## ⚠️ 重要な注意点

### ✅ 安全な編集
```html
<!-- テキストのみ変更OK -->
<h3>医療機器の輸出入・販売</h3>
↓
<h3>あなたのサービス名</h3>
```

### ❌ 危険な編集
```html
<!-- HTMLタグを削除しない -->
<div class="service-card">  ← これを削除しない！
    <h3>サービス名</h3>
</div>
```

## 🆘 トラブル時

| 問題 | 原因 | 解決 |
|------|------|-----|
| ページが表示されない | HTMLが壊れた | バックアップから復元 |
| レイアウトが崩れた | タグを削除した | `<div>` や `</div>` を復元 |
| 色が変わらない | 色コードが間違い | `#` で始まる6文字の色コード |
