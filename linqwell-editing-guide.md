### 4️⃣ サービス内容を変更する

**現在の内容:**
```html
<!-- サービス1 -->
<div class="service-card">
    <div class="service-icon">🩺</div>
    <div class="service-tag">Medical</div>
    <h3>医療機器の輸出入・販売</h3>

**変更方法:**

- `Medical` → サービスのカテゴリ名
- `医療機器の輸出入・販売` → サービスの名前
- `高品質な医療機器...` → サービスの説明

**絵文字の例:**
```

🌍 グローバル
📱 通信
🎓 教育
🏢 企業
📊 分析

## ⚠️ 編集時の注意点

### ❌ やってはいけないこと

1. **HTMLタグを削除しない**
   ```html
   <!-- ❌ 削除してはいけない -->
   <div class="service-card">
   ```

2. **引用符を削除しない**
   ```html
   <!-- ❌ 間違い -->
   <div class=service-card>
   
   <!-- ✅ 正しい -->
   <div class="service-card">
   ```

3. **タグを途中で閉じない**
   ```html
   <!-- ❌ 間違い -->
   <h3>タイトル
   
   <!-- ✅ 正しい -->
   <h3>タイトル</h3>
   ```

### ✅ 安全に編集する方法

- **テキストのみを変更する**（HTMLタグは触らない）
- **編集前にバックアップを取る**
- **1つの変更ごとに保存・確認する**

## 🔍 よくある問題と解決方法

### Q: ページが表示されない

**原因**: HTMLタグが壊れている可能性

**解決方法**:
1. ファイルを閉じる
2. バックアップから復元
3. 1つずつ変更を加え直す

### Q: 色が変わらない

**原因**: 色コードが間違っている可能性

**確認方法**:
- 色コードは`#`で始まる
- 6文字の16進数（0-9, A-F）
- 例：`#2563eb`（正しい）、`#25 63eb`（間違い）

### Q: テキストが改行されない

**原因**: `<br>`タグを使っていない

**解決方法**:
```html
<!-- ❌ 改行されない -->
<p>1行目
2行目</p>

<!-- ✅ 改行される -->
<p>1行目<br>2行目</p>
```

## 💾 保存方法

### Windows
- ファイルを右クリック → **プログラムから開く** → **メモ帳**
- 編集後、`Ctrl+S`で保存

### Mac
- ファイルを右クリック → **このアプリケーションで開く** → **テキストエディット**
- 編集後、`Cmd+S`で保存
